# naavik-opportunity-navigator

Naavik is a proactive, AI-powered, call-based opportunity tracking system designed to help students and job seekers—especially visually impaired and digitally underserved users—access public opportunities such as exams, scholarships, and government jobs.

The system evaluates eligibility and initiates proactive voice notifications when relevant opportunities are published.

## Repository Contents

- requirements.md – Functional and non-functional requirements
- design.md – System architecture and technical design

Built using AWS serverless architecture.
